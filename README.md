# BMO-Companion-Project
This is a personal project for creating "BMO", a character from the Cartoon Network show "Adventure Time."

The goal of this project is to build a BMO that is able to replicate his core functions from the show. As in, serving the purpose of being a portable video game console and general multi-purpose computer. This goal is something I hope to achieve by fully making use of the Raspberry Pi 5's hardware compatibility and software features.
